Punch: a MIDP punch clock application

Punch is a time tracker that runs from your phone or other MIDP device. Create a project, punch in, punch out, and view report on how much time you've spent on a project.

If you're running a MIDP emulator, you can try to launch the latest version of punch.

If you have a MIDP device, try this url: http://wap.dallaway.com/
