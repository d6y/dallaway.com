Access to MUDs via WAP-enabled mobile devices



A long time ago I played MUDs. I don't any more. But if you wanted to access MUDs via a mobile phone how would you do it?

The best way is to have the MUD server provide a WML/WAP interface, tuned to the small screen and limited input of a phone or PDA.

While we wait for that to happen, here's a proposal (PDF; 16k) for a gateway to allow WAP devices to access existing MUDs

If you prefer, you can browse the html version.

There are no known implementations of this proposal.
