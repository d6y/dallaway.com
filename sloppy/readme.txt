SLOPPY README 

The latest version of Sloppy is at  http://www.dallaway.com/sloppy
WHAT IS IT?
Sloppy is a slow proxy.  It deliberately slows the datatransfer between client and server.
WHY?
Sloppy is useful for simulating the experience a dial-upuser would get from your web site (for example).
